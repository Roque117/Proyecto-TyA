# Proyecto-TyA: Sistema de Control y Seguimiento de Rutinas Fitness

![Estado](https://img.shields.io/badge/Estado-Propuesta_3-blue)
![Tecnologia](https://img.shields.io/badge/Stack-Web_Standard-orange)
![Arquitectura](https://img.shields.io/badge/Database-Relacional-336791)

Este sistema consiste en una aplicación web integral diseñada para la gestión personalizada de entrenamientos y el monitoreo del progreso físico. El objetivo es proporcionar una herramienta técnica que permita el registro preciso de métricas corporales, planeación nutricional y evolución de rutinas de ejercicio.

## Descripción del Proyecto

La plataforma ofrece una infraestructura digital para usuarios finales que buscan optimizar su rendimiento físico mediante las siguientes funciones:
* **Gestión de Rutinas:** Programación y seguimiento de series de ejercicio personalizadas.
* **Métricas Corporales:** Registro histórico de peso y medidas para visualización de progreso.
* **Planes Nutricionales:** Módulos de alimentación básica integrados al perfil del usuario.

## Análisis de Factibilidad

Para asegurar la viabilidad del sistema, se realizó una evaluación técnica y operativa detallada:

### Factibilidad Técnica (Alta)
El desarrollo es viable mediante el uso de tecnologías web estándar y el soporte de bases de datos relacionales para la integridad del historial de usuario.

### Factibilidad Económica (Alta)
Se estiman costos de desarrollo moderados con un alto potencial de escalabilidad y monetización mediante modelos de suscripción.

### Factibilidad Operativa (Alta)
Diseño orientado a la experiencia de usuario (UX) intuitiva, permitiendo una adopción rápida con mínima capacitación técnica.

### Factibilidad Legal (Media-Alta)
El sistema integra protocolos de manejo de datos personales y avisos de responsabilidad legal para garantizar el cumplimiento normativo.
